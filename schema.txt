Redis structures used by this application

<name> - Bot data
<id> - Message data

bots - Index of bot names
messages - Index of message id's
<name>:messages - Index of message id's for a specific bot

<token> - token:<name> history
 
following:<name> - Index of bot names that bot <name> is following
followers:<name> - Index of bot names that are following bot <name>
